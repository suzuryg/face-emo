# 表情

本工具通过以下三项组合条件来指定表情的变化方式：
- 手的种类  
- 手势类型  
- 是否一致  

|<center>条件</center>|<center>行为</center>|
|:-:|:-|
|「左手が」「Fist」「と一致」|当左手为 Fist 时切换表情（不考虑右手状态）|
|「両手が」「Open」「と一致」|当双手均为 Open 时切换表情|
|「片手が」「Point」「と一致」|当任意一只手为 Point 时切换表情（双手同时为 Point 时不会变化）|
|「片手または両手が」「Victory」「と一致」|当任意一只手为 Victory 时切换表情（双手为 Victory 时也会变化）|

可以组合多个条件共同使用。

|<center>条件</center>|<center>行为</center>|
|:-:|:-|
|「片手が」「Rock」「と一致」 + 「片手が」「Gun」「と一致」|当左手为 Rock 且右手为 Gun，或左手为 Gun 且右手为 Rock 时切换表情|

当存在多个表情时，会**从上到下依次判定**。

- 例如以下两个表情同时存在时，第一项优先：当左手为 Fist 时会使用第一个表情  
    - 「左手が」「Fist」「と一致」  
    - 「片手が」「Fist」「と一致」
- 表情顺序可通过拖动调整
- 当前哪个表情优先可在「手势表」中查看

![表情](expression.png)

每个表情可修改以下设置：

|<center>项目名</center>|<center>设置内容</center>|
|:-:|:-|
|启用视线追踪|设置是否启用伪视线追踪。与眨眼设置独立。|
|启用眨眼|设置是否允许眨眼。若表情中眼睛为闭合状态，请禁用眨眼。|
|启用嘴型同步|设置是否启用嘴型同步。<br/>若想防止说话时表情崩坏，请禁用嘴型同步或使用口部变形取消功能。|
|启用口部变形取消|在说话时，将特定 Shape Key 的值恢复为默认值（场景中设置的值），以防止崩坏。<br/>需在检视器中「口変形キャンセル用シェイプキー」处指定要恢复的 Shape Key。|
|使用左扳机|根据左手扳机的按压量变化表情。请额外设置按下扳机时的
