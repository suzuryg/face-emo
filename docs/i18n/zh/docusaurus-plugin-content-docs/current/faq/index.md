# 常见问题FAQs

## 改变手势后表情没有变化

请确认以下几点。

|<center>检查点</center>|<center>解决方法</center>|
|:-|:-|
| Expression菜单中未添加“FaceEmo” | 按照[教程](../tutorials/)的步骤，将表情菜单应用到头像上。|
| “FaceEmo”菜单中的表情模式未正确选择 | 选择合适的表情模式。|
| “FaceEmo”→“表情选择”菜单已打开 | 关闭“表情选择”菜单（使用“返回”）。<br/>（请确认左右手的Expression菜单）|
| “FaceEmo”→“表情选择”→“表情固定ON”已启用 | 禁用“表情固定ON”。|
| “FaceEmo”→“设置”→“表情固定ON”已启用 | 禁用“表情固定ON”。|

参考：[选项功能介绍 > 固定表情](../optional-functions/emote-lock/)

## 可以与头像原有设置的表情同时使用吗？

为了防止表情干扰，头像原有设置的表情会被禁用。  
请将类似的表情设置添加到本工具的表情菜单中。

## 如何删除头像中的表情菜单？

请删除头像中添加的“FaceEmoPrefab”。  
参考：[教程 > 创建简单表情菜单](../tutorials/simple-menu/)

## 可以再次显示“不要再显示”勾选的对话框吗？

可以通过检查器的“编辑器设置”更改对话框的显示状态。

## UI显示异常怎么办？

如果出现如下问题，切换到“播放模式”再返回“编辑模式”可能会解决问题。

- 各视图宽度不正确调整
- 表情缩略图无法绘制

## 想要自动将表情选择菜单的内容整理进文件夹

请启用“选项设置” -> “角色应用设置” -> “表情超8个时自动生成文件夹”。
