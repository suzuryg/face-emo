# 防止嘴型同步导致表情崩坏

当表情本身带有张嘴动作时，如果同时启用了嘴型同步（LipSync），可能会导致表情**崩坏**。  
可以通过以下任意方法进行处理：
- [禁用嘴型同步](#禁用嘴型同步)
- [在嘴型同步时取消口部变形](#在嘴型同步时取消口部变形)

## 禁用嘴型同步

在每个表情设置中取消勾选「リップシンク有効」（启用嘴型同步），即可让该表情不再执行嘴型同步。

![禁用嘴型同步](disable_lipsync.png)

---

## 在嘴型同步时取消口部变形

在每个表情设置中勾选「口変形キャンセル」（取消口部变形），即可启用该功能。  
启用后，在说话时可将特定的**口部 Shape Key**恢复为默认值（即场景中设置的值）。

![启用口部变形取消](enable_mouth_morph_cancel.png)

需要恢复到默认值的 Shape Key 可在检视器中的「口変形キャンセル用シェイプキー」处设置。

![设置口部变形取消 Shape Key](add_mouth_morph.png)

:::caution
若口部变形取消未能正常工作，请确认所有与口部相关的 Shape Key 是否都已添加。  
若你使用的是 **MMD 用 Shape Key** 控制口部变形，也请一并添加进去。
:::
