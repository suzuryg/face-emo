部分可选功能需要使用 FaceEmo 的设置界面。请在层级视图（Hierarchy）中选择「FaceEmo」对象，并显示如下界面。

![选择表情菜单](select_menu.png)

* [将表情菜单应用到多个 Avatar](apply-multiple/)
* [固定表情](emote-lock/)
* [根据扳机按压量变化表情](use-trigger/)
* [防止嘴型同步导致表情崩坏](mouth-morph/)
* [使粒子与表情对应](additional-toggles/)
* [使兽耳或呆毛的动作与表情对应](additional-transforms/)
* [设置被抚摸时的表情](contact-override/)
* [在 Expression Menu 中修改表情设置](setting-menu/)
* [从自动备份中恢复表情菜单](auto-backup/)
