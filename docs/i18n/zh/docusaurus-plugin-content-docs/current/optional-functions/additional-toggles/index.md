# 将粒子与面部表情对应  

在“添加面部表情对象（Toggle）”中添加对象后，可以同时控制表情和对象的开关状态。  

## 添加对象  

在检视器中打开“添加面部表情对象（Toggle）”，然后添加希望控制的对象。  

![添加对象](add_objects.png)  

:::tip  
在这里添加的对象，当未在动画中进行控制时，将恢复到默认状态。  
因此，不需要为恢复到默认状态而制作动画。  
:::

---

## 在面部表情动画中设置对象的ON/OFF  

添加到“添加面部表情对象（Toggle）”的对象，将可以在表情编辑器中使用。  
像形状键（Shape Key）一样，将对象加入动画中并设置ON/OFF状态。  
请注意，粒子的ON/OFF状态不会反映在动画预览中。  

![创建面部表情动画](add_key.png)
